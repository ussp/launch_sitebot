## Brand Culture & Values

### Organizational Feelings (18 Core Beliefs)

#### People & Team
1. We believe if you build great people, they will build a great company.
2. We believe it takes a team to provide the best experiences; every position and every player is valued.
3. We believe in ongoing recognition and personal growth of our team.

#### Attitude & Behavior
4. We believe you should throw laughter, smiles and kindness around like confetti.
5. We believe happiness is the key to success. If you love what you are doing, you will be successful.

7. We believe you are always responsible for how you act, no matter how you feel.
8. We believe in being kind and treating others like you want to be treated.

#### Learning & Growth
9. We believe you can learn great things from your mistakes when you aren't busy denying them.
10. We believe success is the result of ambition, hard work and learning from failure.
11. We believe the world is changed by our example, not our opinion.
12. We believe when we focus on possibilities, we'll have more opportunities.

#### Guest Experience
13. We believe in treating our customers like long lost friends that we are excited to see.
14. We believe families deserve to have an amazing guest experience, while creating unique memories that they will cherish.

#### Community
15. We believe in being a great neighbor and giving back to our community.

#### Operations
16. We believe in safety. Safety for our team and safety for our guests.
17. We believe in providing a clean, organized facility at all times.
18. We believe even the smallest detail can have a big impact.

## Asset Selection Logic

### Logo Selection by Background

```python
def select_logo(background: str, format: str = "png", single_color: bool = False) -> str:
    """
    Select the appropriate logo file based on background and format.

    Args:
        background: "light" or "dark"
        format: "png" (web) or "eps" (print)
        single_color: True for single-color version

    Returns:
        Relative path to the logo file
    """
    logos = {
        "light": {
            "full_color": {
                "png": "assets/logos/LaunchFE_Logo_2024_Black.png",
                "eps": "assets/logos/LaunchFE_Logo_2024_Black.eps"
            },
            "single_color": {
                "png": "assets/logos/LaunchFE_Logo_2024-OneColorBlack.png",
                "eps": "assets/logos/LaunchFE_Logo_2024-OneColorBlack.eps"
            }
        },
        "dark": {
            "full_color": {
                "png": "assets/logos/LaunchFE_Logo_2024-White.png",
                "eps": "assets/logos/LaunchFE_Logo_2024-White.eps"
            },
            "single_color": {
                "png": "assets/logos/LaunchFE_Logo_2024-OneColorWhite.png",
                "eps": "assets/logos/LaunchFE_Logo_2024-OneColorWhite.eps"
            }
        }
    }

    color_type = "single_color" if single_color else "full_color"
    return logos[background][color_type][format]
```

## Validation Rules for Agents

### Logo on Background Validation

```python
LOGO_BACKGROUND_RULES = {
    "white": ["black", "launch_green", "launch_yellow"],
    "black": ["white", "launch_green", "launch_yellow"],
    "launch_green": ["black", "white", "launch_yellow"],
    "launch_yellow": ["black", "launch_green", "white"]
}

def is_valid_logo_on_background(background_color: str, logo_color: str) -> bool:
    """Check if a logo color is valid on a given background."""
    return logo_color in LOGO_BACKGROUND_RULES.get(background_color, [])
```

### Font for Element Validation

```python
FONT_RULES = {
    "headline": "Futura Bold",
    "sub_header": "Futura PT Bold",
    "body": "Futura Medium",
    "button": "Futura Bold",
    "fine_print": "Futura Medium"

def get_font_for_element(element_type: str) -> str:
```

### Gradient Validation

```python
VALID_GRADIENTS = [
    ("#5CBA47", "#F4E501"),  # Green -> Yellow
    ("#B035C8", "#EA3D6F"),  # Purple -> Pink
    ("#F4E501", "#FF9307"),  # Yellow -> Orange
    ("#5CBA47", "#1E8FD7"),  # Green -> Blue
    ("#FF9307", "#EF4036"),  # Orange -> Red
    ("#B035C8", "#1E8FD7"),  # Purple -> Blue
]

def is_valid_gradient(color1: str, color2: str) -> bool:
    """Check if a gradient combination is approved."""
    return (color1, color2) in VALID_GRADIENTS or (color2, color1) in VALID_GRADIENTS
```
